## instillation

> remove `sudo` from commands, if you are installing on [termux](https://play.google.com/store/apps/details?id=com.termux)

Ö† `sudo` `apt update`

Ö† `sudo` `apt upgrade`

Ö† `sudo` `apt install git`

Ö† `git clone https://github.com/arNparad0x/thm2md`

Ö† `cd thm2md`

Ö† `chmod +x *`

>or
Ö† `chmod +x thm2md requirements.sh`

Ö†`sudo` `./requirements.sh`

### to run it from any location:
 <p id="path">add it to your $PATH</p>
 
>>linux:

Ö†`cp thm2md /usr/local/bin/`

>>termux:

Ö†`cp thm2md ~/../usr/bin/`

### --help / -h for help
* Ö† thm2md --help
>or
* Ö† thm2md -h
